# 事件日历应用 - SPEC.md

## 1. 项目概述

- **项目名称**: 事件日历 (EventCalendar)
- **项目类型**: Android 原生应用
- **核心功能**: 用户可以在日历中记录事件，查看过去的记录，并统计事件发生频次

## 2. 技术栈

- **语言**: Kotlin 1.9.x
- **最低 SDK**: 26 (Android 8.0)
- **目标 SDK**: 34 (Android 14)
- **UI 框架**: Jetpack Compose + Material 3
- **架构**: MVVM + Clean Architecture
- **本地数据库**: Room
- **依赖注入**: Hilt
- **异步操作**: Kotlin Coroutines + Flow
- **日历组件**: 自定义 Compose 日历视图

## 3. 功能列表

### 3.1 日历主界面
- 显示月历视图，支持月份切换
- 在日期上显示事件标记点
- 点击日期查看当天事件列表

### 3.2 事件管理
- 添加事件：选择日期、时间、输入事件标题和备注
- 事件支持设置提醒（可选）
- 查看事件详情
- 删除事件

### 3.3 历史记录
- 按日期查看所有已记录的事件
- 按事件类型（标题）筛选历史记录
- 事件列表显示时间线

### 3.4 频次统计
- 按事件标题统计出现次数
- 显示每个事件的历史记录列表
- 支持查看今日、本周、本月、自定义时间段的统计

## 4. UI/UX 设计方向

### 4.1 整体风格
- Material Design 3
- 简洁清爽的界面风格
- 浅色主题为主

### 4.2 颜色方案
- 主色: 蓝色系 (#2196F3)
- 强调色: 橙色 (#FF9800) 用于标记有事件的日期
- 背景: 白色/浅灰色

### 4.3 布局
- 底部导航栏: 日历 | 历史 | 统计
- 日历页面: 顶部月份切换 + 月历视图 + 事件列表
- 添加事件: 底部弹窗表单
- 历史/统计: 列表视图

### 4.4 交互
- 点击日期弹出当天事件列表
- 长按日期快速添加事件
- 右滑删除事件
